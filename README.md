# 💻 CodeMaster

An AI-powered Programming Quiz Platform built using Python, Streamlit, and SQLite.

## 🚀 Features

- User Login & Signup
- Python, Java, C, SQL Quizzes
- AI Explanations
- Leaderboard
- Progress Tracking
- PDF Certificate

## 🛠 Tech Stack

- Python
- Streamlit
- SQLite
- HTML
- CSS
